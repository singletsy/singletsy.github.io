this指向

Veröffentlicht am 2017-07-19 | in javascript

this指向

通过函数名()直接调用：this指向window

通过对象.函数名()调用的：this指向这个对象

函数作为数组的一个元素，通过数组下标调用的：this指向这个数组

函数作为window内置函数的回调函数调用：this指向window（ setInterval setTimeout 等）

函数作为构造函数，用new关键字调用时：this指向新new出的对象

远程仓库管理操作

Veröffentlicht am 2017-07-19 | in git

分支相关的命令

创建分支

git branch 分支名称

查看所有分支

git branch

切换到指定分支

git checkout 分支名

合并分支

git merge 分支名
将指定的分支 合并到 当前的分支

删除分支

git branch -d 分支名称

删除分支的时候，如果分支修改的内容没有被合并，那么会提示不能删除，要删除需要合并，或者使用-D

创建并切换到新的分支

git checkout -b 分支名

在分支进行切换之前，要对所有的内容进行一次add 和commit操作

master分支是在仓库初始化之后，第一次提交完成的时候才出现的！

setTimeout

Veröffentlicht am 2016-02-10

setTimeout， 至少在指定的时间后执行指定的代码！

1.js是单线程的，同时只能干一件事

2.js中的任务，分为两类

1. 主要任务(for循环)
2. 次要任务(setTimeout setInterval 事件)

3.事件队列，js代码在执行主要任务的时候，会将所有遇到的次要任务放到一个叫做事件队列的容器当中去，当主要的任务执行完毕之后，才会去这个容器中执行次要的任务
